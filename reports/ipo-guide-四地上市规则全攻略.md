## 一、上海证券交易所主板

### 1.1 法规依据

| 文件名 | 文号 | 发布时间 |
|--------|------|----------|
| 《上海证券交易所股票上市规则》 | 上证发〔2025〕51号 | 2025年4月 |
| 《首次公开发行股票注册管理办法》 | 证监会令第205号 | 2023年2月 |
| 《公司法》 | 2023年修订 | 2023年12月 |

> **来源**：上海证券交易所官网 www.sse.com.cn

### 1.2 发行条件

1. **持续经营满3年**
   - 有限责任公司按账面净资产折股整体变更为股份有限公司的，可从有限责任公司成立之日起计算

2. **会计基础规范，内控制度健全**
   - 财务报告由注册会计师出具无保留意见审计报告
   - 内控鉴证报告为无保留结论

3. **业务完整，具有直接面向市场独立持续经营能力**
   - 资产完整，业务/人员/财务/机构独立
   - 不存在对发行人构成重大不利影响的同业竞争
   - 最近3年内主营业务和董事、高级管理人员没有重大不利变化
   - 最近3年实际控制人没有发生变更

4. **经营合法合规**
   - 符合国家产业政策
   - 最近3年不存在贪污、贿赂、侵占财产、挪用财产或破坏社会主义市场经济秩序的刑事犯罪
   - 不存在欺诈发行、重大信息披露违法

### 1.3 上市财务标准（市值+财务指标）

| 标准 | 净利润 | 营业收入 | 现金流量净额 | 预计市值 |
|------|--------|----------|--------------|----------|
| **标准一** | 最近3年均为正，累计≥2亿元，最近1年≥1亿元 | 累计≥15亿元 | 累计≥2亿元 | - |
| **标准二** | 最近1年为正 | 最近1年≥6亿元 | 最近3年累计≥2.5亿元 | ≥50亿元 |
| **标准三** | 最近1年为正 | 最近1年≥10亿元 | - | ≥100亿元 |

> **说明**：净利润、营业收入、现金流量净额均以扣除非经常性损益前后孰低为准

### 1.4 特殊企业上市标准

#### 红筹企业

| 类型 | 市值 | 营业收入 | 其他 |
|------|------|----------|------|
| 已境外上市 | ≥2000亿元 | - | - |
| 未境外上市 | ≥200亿元 | 最近1年≥30亿元 | 自主研发、国际领先技术 |
| 未境外上市（估值） | ≥100亿元 | 最近1年≥5亿元 | 营收增长快，自主技术 |

#### 表决权差异安排（AB股）

| 预计市值 | 净利润要求 |
|----------|------------|
| ≥200亿元 | 最近1年为正 |
| ≥100亿元 | 最近1年为正且营业收入≥10亿元 |

## 二、上海证券交易所科创板

### 2.1 法规依据

| 文件名 | 文号 |
|--------|------|
| 《上海证券交易所科创板股票上市规则》 | 上证发〔2025〕60号 |
| 《科创属性评价指引（试行）》 | 证监会公告〔2025〕9号 |

> **来源**：上海证券交易所官网 www.sse.com.cn

### 2.2 科创板定位

**支持和鼓励**：
- 新一代信息技术
- 高端装备
- 新材料
- 新能源
- 生物医药
- 高端服务
- 节能环保
- 新能源汽车
- 人工智能
- 量子科技
- 脑机接口
- 通用智能
- 无人系统
- 低空经济
- 商业航天

### 2.3 上市财务标准

| 标准 | 市值 | 研发投入 | 营业收入 | 现金流 | 核心技术 |
|------|------|----------|----------|--------|----------|
| **标准一** | ≥10亿元 | - | 最近1年≥1亿元 | 最近3年累计≥2亿元 | - |
| **标准二** | ≥15亿元 | 最近3年累计≥5亿元 | 最近1年≥2亿元 | - | - |
| **标准三** | ≥20亿元 | 最近3年累计≥2亿元 | 最近1年≥3亿元 | - | 核心技术市场化应用 |
| **标准四** | ≥30亿元 | - | 最近1年≥3亿元 | - | 自主研发、国际领先 |
| **标准五** | ≥40亿元 | - | 主营业务收入≥5亿元 | - | 自主研发、国际领先 |

### 2.4 科创属性评价

**常规指标**：
- 最近3年研发投入占营业收入比例≥5%，或最近3年研发投入金额累计≥8000万元
- 最近3年营业收入复合增长率≥25%，或最近1年营业收入金额≥3亿元
- 发行后股本总额≥3000万元

**例外情形**：
- 核心技术属于国家战略方向，并形成核心产品
- 依靠核心技术形成的主要产品已实现进口替代

## 四、香港联交所主板

### 4.1 法规依据

| 文件名 |
|--------|
| 《主板上市规则》（Main Board Listing Rules） |
| 《公司条例》（Companies Ordinance） |
| 《证券及期货条例》（Securities and Futures Ordinance） |

> **来源**：香港联交所官网 www.hkex.com.hk

### 4.2 主板定位

为大型、成熟企业提供融资平台，分为：
- **一般企业**：有盈利或特定财务要求
- **H股企业**：中国内地企业
- **红筹企业**：境外注册、中资控股

### 4.3 上市财务标准

| 标准 | 股东应占盈利 | 市值 | 收入 | 现金流量 |
|------|--------------|------|------|----------|
| **标准一** | 最近3年累计≥5000万港元，最近1年≥2000万港元 | ≥5亿港元 | - | - |
| **标准二** | - | ≥40亿港元 | 最近1年≥6亿港元 | 最近3年累计≥1亿港元 |
| **标准三** | - | ≥20亿港元 | 最近1年≥5亿港元 | 最近3年累计≥1亿港元 |

### 4.4 营业记录要求

- **一般情况**：3年营业记录
- **特殊情况**：可接纳不足3年（需符合特定条件）
- **管理层要求**：最近1年管理层不变
- **拥有权要求**：最近1年控制权不变

### 4.5 特殊上市途径

#### 18A章（生物科技）
- 核心产品已通过一期临床试验
- 上市时市值≥15亿港元
- 聆讯时拥有至少6个月现金储备

#### 18C章（特专科技）
- 收入≥2.5亿港元（最近一年）
- 上市时市值≥80亿港元
- 研发投入占比≥15%

#### SPAC（特殊目的收购公司）
- 最低集资额10亿港元
- 发起人资质要求
- 24个月完成并购（可延长12个月）

### 4.6 港股上市优势

| 优势 | 说明 |
|------|------|
| 国际化 | 面向全球投资者 |
| 估值 | 部分行业估值高于A股 |
| 再融资 | 增发、配股便捷 |
| 汇率 | 港币与美元挂钩 |
| 退市 | 无A股ST制度 |

### 5.3 审核周期对比

| 市场 | 审核周期（参考） |

### 5.5 核心优势对比

| 市场 | 核心优势 |

## 七、实务建议

### 7.1 如何选择上市地？

| 企业特点 | 推荐市场 |
|----------|----------|
| 传统行业、盈利稳定 | 上交所主板 |
| 科技创新、未盈利 | 科创板 |
| 成长型、创新业态 | 深交所创业板 |
| 国际化布局、美元基金 | 港交所主板 |
| 生物医药早期 | 港交所18A |
| 特专科技 | 港交所18C |

### 7.2 注意事项

1. **A股**：需提前规划社保、公积金合规；关注关联交易
2. **港股**：保荐人资质重要；国际路演成本高
3. **两地上市**：可A+H同步，需协调审核进度
